# Platform Insights Reading Guide

## Overview

This guide walks you through accessing and interpreting native analytics on major social platforms. Use this to gather accurate data for performance analysis.

---

## Facebook / Meta Business Suite

### Accessing Insights

**Path**: Facebook Page → Insights (left sidebar) → Overview

**Alternative**: Meta Business Suite → Insights → [Select Page]

### Key Sections and Metrics

#### 1. Overview Tab
**Location**: First screen after opening Insights

**Key Metrics Available**:
- Page Views (daily, weekly, monthly)
- Page Likes and Unlikes
- Reach (organic vs paid)
- Post Engagement (reactions, comments, shares)
- Page Followers (net growth)

**What to Look For**:
- Daily reach trends (spikes indicate viral content)
- Engagement rate trends (consistency matters)
- Follower growth velocity (positive = healthy)

#### 2. Content Tab
**Location**: Insights → Content

**Key Data Points**:
- Top performing posts by engagement
- Post types breakdown (link, photo, video, status)
- Publishing time vs engagement correlation
- Content reach and impressions

**What to Look For**:
- Which content format gets most engagement
- Optimal posting times for your audience
- Topics that resonate most

#### 3. People Tab
**Location**: Insights → People

**Demographic Data**:
- Age and gender distribution
- Top locations (cities, countries)
- Language preferences
- Active times (when followers are online)

**What to Look For**:
- Alignment between target audience and actual followers
- Geographic opportunities
- Optimal posting schedule based on online times

#### 4. Page Performance
**Location**: Insights → Performance

**Metrics Breakdown**:
- Engagement rate by post type
- Response rate and time (for Messages)
- Actions on Page (button clicks)
- Video performance (3-second views, minutes viewed)

**What to Look For**:
- CTA effectiveness (button click rates)
- Video retention patterns
- Customer service efficiency

### Exporting Facebook Data

**Method 1: Manual Export**
1. Go to Insights → Overview
2. Click "Export Data" (top right)
3. Select date range and data type
4. Choose format (Excel or CSV)
5. Download and open file

**Method 2: Data for Analysis Paste**
Copy these metrics for quick analysis:
```
Period: [Start Date] - [End Date]
Page Likes: [NUMBER]
Page Reach: [NUMBER]
Post Engagement: [NUMBER]
Top Post: [Title/Description] - [Engagement Count]
```

## Instagram Insights

### Accessing Insights

**Path**: Instagram Profile → Menu (three lines) → Insights

**Requirements**: Must have Business or Creator account

### Key Sections and Metrics

#### 1. Overview Tab
**Location**: Insights home screen

**Key Metrics Available**:
- Accounts Reached (last 7/30 days)
- Accounts Engaged (last 7/30 days)
- Total Followers (net change)
- Content Interactions (likes, comments, saves, shares)

**What to Look For**:
- Reach vs engagement ratio (5:1 is healthy)
- Follower growth consistency
- High-value interactions (saves and shares)

#### 2. Content Tab
**Location**: Insights → Content You Shared

**Subcategories**:
- **Feed Posts**: Individual post performance
- **Stories**: Story reach and interactions
- **Reels**: Views, engagement, saves
- **Live**: Peak concurrent viewers, comments

**What to Look For (Feed)**:
- Save rate (indicates valuable content)
- Share rate (viral potential)
- Profile visits from post (traffic driver)

**What to Look For (Reels)**:
- Plays vs Accounts Reached (higher = algorithm boost)
- Average watch time (50%+ is good)
- Shares and saves (high-value signals)

**What to Look For (Stories)**:
- Completion rate (viewers who watched all slides)
- Forward taps (content skipped)
- Exit rate (when users leave story)
- Replies and sticker interactions

#### 3. Audience Tab
**Location**: Insights → Audience

**Demographic Data**:
- Follower growth trends
- Top locations (cities)
- Age and gender breakdown
- Most active times (hourly breakdown)

**What to Look For**:
- Peak activity windows for posting
- Audience demographic shifts over time
- Geographic concentration opportunities

#### 4. Individual Post Insights
**Access**: Tap any post → View Insights

**Detailed Metrics**:
- Impressions (From Home, From Hashtags, From Explore, etc.)
- Engagement breakdown (likes, comments, saves, shares)
- Profile Activity (profile visits, follows from post)
- Navigation (forward, back, next story, exit)

**What to Look For**:
- Discovery sources (optimize hashtags if low)
- Save-to-like ratio (>0.2 is excellent)
- Profile visit rate (indicates compelling content)

### Exporting Instagram Data

**Method 1: Screenshots**
Instagram doesn't offer native export, so:
1. Screenshot key metrics screens
2. Use image-to-text OCR for data extraction
3. Manually compile into spreadsheet

**Method 2: Data for Analysis Paste**
Copy these metrics for quick analysis:
```
Period: [Last 7/30 days]
Accounts Reached: [NUMBER]
Accounts Engaged: [NUMBER]
New Followers: [NUMBER]
Top Post: [Description] - [Likes] likes, [Comments] comments, [Saves] saves
Reels Avg Views: [NUMBER]
Story Completion Rate: [PERCENTAGE]
```

## TikTok Analytics

### Accessing Analytics

**Path**: Profile → Menu (three lines) → Business Suite → Analytics

**Requirements**: Must switch to Business or Creator account (Settings → Manage Account)

### Key Sections and Metrics

#### 1. Overview Tab
**Location**: Analytics home screen

**Key Metrics Available** (7/28/60 days):
- Video Views
- Followers (net change)
- Profile Views
- Total Likes

**What to Look For**:
- View velocity (sudden spikes = viral content)
- Follower conversion rate (views to followers)
- Profile visit rate (content interest indicator)

#### 2. Content Tab
**Location**: Analytics → Content

**Post Performance Metrics**:
- Total Views per video
- Average Watch Time
- Watched Full Video (completion rate)
- Likes, Comments, Shares
- Traffic Source Type (For You, Following, Search, etc.)

**What to Look For**:
- Completion rate (50%+ is algorithm-friendly)
- Share rate (1%+ has viral potential)
- For You Page (FYP) traffic percentage (higher = algorithm boost)

**Trending Videos Section**:
- Top 10 videos by views (last 7 days)
- Performance comparison against account average

**What to Look For**:
- Common elements in top performers (hooks, music, format)
- Outliers (why one video outperformed others)

#### 3. Followers Tab
**Location**: Analytics → Followers

**Audience Insights**:
- Gender distribution
- Top territories (countries)
- Follower activity (daily and hourly breakdown)
- Videos your followers watched
- Sounds your followers listened to

**What to Look For**:
- Peak activity times for posting
- Content preferences of existing audience
- Demographic alignment with target market

#### 4. Live Analytics
**Location**: Analytics → LIVE (if live streaming enabled)

**Live Stream Metrics**:
- Peak viewers
- New followers from live
- Diamonds earned (if monetized)
- Total viewers and duration

### Exporting TikTok Data

**Method 1: Native Export**
1. Go to Analytics → Overview
2. Tap "Export" (top right)
3. Select date range
4. Download CSV or Excel file
5. Open in spreadsheet software

**Method 2: Data for Analysis Paste**
Copy these metrics for quick analysis:
```
Period: [Last 7/28 days]
Total Video Views: [NUMBER]
New Followers: [NUMBER]
Profile Views: [NUMBER]
Top Video: [Description/Link] - [Views] views, [Completion Rate]%
Avg Engagement Rate: [PERCENTAGE]
```

## YouTube Studio Analytics

### Accessing Analytics

**Path**: YouTube Studio → Analytics (left sidebar)

**Alternative**: youtube.com/analytics

### Key Sections and Metrics

#### 1. Overview Tab
**Location**: Analytics home screen

**Key Metrics Available** (last 7/28/90/365 days):
- Views
- Watch Time (hours)
- Subscribers (net change)
- Estimated Revenue (if monetized)

**Real-Time Activity**:
- Views and watch time (last 48 hours)
- Top videos currently being watched

**What to Look For**:
- Watch time trends (algorithm priority)
- Subscriber conversion rate (views to subs)
- Revenue per 1,000 views (RPM) if monetized

#### 2. Reach Tab
**Location**: Analytics → Reach

**Key Metrics**:
- Impressions (how many saw thumbnail)
- Impressions Click-Through Rate (CTR)
- Views and Unique Viewers
- Traffic Sources (Browse, Suggested, Search, External, etc.)
- Top Thumbnails by impressions
- Top YouTube Search Terms

**What to Look For**:
- CTR (4-10% is good, >10% is excellent)
- Traffic source distribution (diversification is healthy)
- Search terms (content gap opportunities)
- Thumbnail effectiveness (test and iterate)

#### 3. Engagement Tab
**Location**: Analytics → Engagement

**Key Metrics**:
- Average View Duration
- Average Percentage Viewed
- Likes, Dislikes (if visible), Comments, Shares
- Audience Retention Graph (by video)
- End Screen Element Clicks
- Card Clicks

**What to Look For**:
- Retention curve drop-off points (optimize intro)
- Average view percentage (50%+ is strong)
- High-retention videos (replicate format)
- End screen click rate (CTA effectiveness)

#### 4. Audience Tab
**Location**: Analytics → Audience

**Key Metrics**:
- Unique Viewers
- Returning Viewers vs New Viewers
- Subscribers vs Non-Subscribers watching
- When Your Viewers Are on YouTube (heatmap)
- Top Geographies, Age, Gender
- Suggested Videos (what else your audience watches)

**What to Look For**:
- Subscriber loyalty (% of views from subs)
- Optimal upload times (based on activity heatmap)
- Audience interests (content expansion ideas)
- Subscriber growth sources (which videos convert)

#### 5. Shorts Analytics
**Location**: Analytics → Shorts (if published Shorts)

**Shorts-Specific Metrics**:
- Views
- Swipe Away Rate
- Engagement (likes, comments, shares)
- Subscriber impact (net change from Shorts)

**What to Look For**:
- Low swipe rate (<70% is good)
- Conversion to channel views (Shorts as discovery)
- Comment sentiment and themes

### Exporting YouTube Data

**Method 1: Native Export**
1. Go to Analytics → Advanced Mode
2. Click "Export Current View" (top right)
3. Choose format (CSV, Excel, Google Sheets)
4. Download file

**Method 2: Data for Analysis Paste**
Copy these metrics for quick analysis:
```
Period: [Last 28 days]
Total Views: [NUMBER]
Watch Time: [HOURS] hours
New Subscribers: [NUMBER]
Avg View Duration: [MINUTES:SECONDS]
CTR: [PERCENTAGE]%
Top Video: [Title] - [Views] views, [Avg %] viewed
Traffic Sources: Browse [%], Search [%], Suggested [%]
```

## Tips for Accurate Data Collection

1. **Consistent Timeframes**: Always compare equal periods (7 days to 7 days)
2. **Note Anomalies**: Record external events affecting data (campaigns, holidays, viral moments)
3. **Screenshot Everything**: Platforms change interfaces; screenshots preserve context
4. **Export Regularly**: Don't rely on platforms retaining historical data indefinitely
5. **Cross-Reference**: Verify metrics match across mobile and desktop versions
6. **Time Zone Awareness**: Ensure analytics timezone matches your target audience
7. **Organic vs Paid**: Separate paid promotion data from organic performance when possible

## Common Access Issues and Solutions

**Issue**: "Insights Not Available Yet"
**Solution**: Need 100+ followers (Instagram) or 7+ days of data

**Issue**: Missing metrics in export
**Solution**: Some metrics only available in-app, not in exports

**Issue**: Data discrepancies between mobile and desktop
**Solution**: Use desktop for accuracy; mobile rounds numbers

**Issue**: Can't access historical data beyond 90 days
**Solution**: Export monthly and maintain your own archive

**Issue**: Analytics delayed or not updating
**Solution**: Wait 24-48 hours; platforms batch-process data
