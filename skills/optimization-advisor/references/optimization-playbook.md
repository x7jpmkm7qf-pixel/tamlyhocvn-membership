# Optimization Playbook

Actionable optimization patterns using IF-THEN decision trees for common digital business challenges.

## Traffic Optimization

### IF Reach Dropping THEN

**Diagnosis Criteria:**
- Weekly reach decreased by >15%
- Follower growth stagnant or negative
- Impressions declining consistently

**Root Cause Possibilities:**
- Algorithm disfavor due to engagement drop
- Content format shift (platform favoring new formats)
- Posting frequency inconsistency
- Audience fatigue with content type

**Action Steps:**
1. Audit last 30 posts → identify engagement patterns → double down on top-performing formats
2. Test new content formats (Reels, Stories, Carousels) → measure engagement lift → allocate 30% capacity to winners
3. Implement consistent posting schedule → minimum 4x/week → track follower retention rate

**Expected Impact:** 20-40% reach recovery within 4 weeks
**Measurement:** Weekly reach metrics, follower growth rate, engagement rate per post

### IF Impressions High but Engagement Low THEN

**Diagnosis Criteria:**
- Impression-to-engagement ratio <2%
- High views but low likes/comments/shares
- Watch time <30% of video length

**Root Cause Possibilities:**
- Content misaligned with audience expectations
- Weak hooks/thumbnails (attract wrong audience)
- Call-to-action unclear or missing
- Content quality below platform standards

**Action Steps:**
1. A/B test first 3 seconds of content → measure completion rate → optimize hooks
2. Analyze audience demographics vs content theme → realign messaging to core audience
3. Add explicit CTAs every 30 seconds → test different CTA types → measure action rate

**Expected Impact:** 3-5x engagement rate increase within 2 weeks
**Measurement:** Engagement rate, comment quality, share rate

### IF Traffic Sources Imbalanced THEN

**Diagnosis Criteria:**
- >70% traffic from single source
- Low diversification score
- Vulnerable to algorithm changes

**Root Cause Possibilities:**
- Over-reliance on one platform
- Lack of cross-promotion strategy
- Email list underdeveloped
- SEO neglected

**Action Steps:**
1. Build email list → offer lead magnet → send weekly value content → measure open rates
2. Optimize top 10 pages for SEO → target long-tail keywords → track organic growth
3. Cross-promote on 2 new platforms → repurpose existing content → allocate 20% effort

**Expected Impact:** 30-50% traffic diversification within 8 weeks
**Measurement:** Traffic source breakdown, channel resilience score

## Conversion Optimization

### IF Bounce Rate >70% THEN

**Diagnosis Criteria:**
- Users leave without interaction
- Avg session duration <30 seconds
- High exit rate on landing pages

**Root Cause Possibilities:**
- Slow page load speed (>3 seconds)
- Misleading ad copy vs landing page
- Poor mobile experience
- Unclear value proposition

**Action Steps:**
1. Run PageSpeed Insights → fix Core Web Vitals → compress images → lazy load content
2. Audit message match between ads and landing page → align copy/design → A/B test variants
3. Test mobile experience → simplify navigation → ensure CTA above fold → measure mobile conversion rate

**Expected Impact:** Reduce bounce rate to 40-50% within 2 weeks
**Measurement:** Bounce rate, avg session duration, pages per session

### IF Checkout Abandonment High (>60%) THEN

**Diagnosis Criteria:**
- Users add to cart but don't complete purchase
- Drop-off at payment step
- High cart abandonment rate

**Root Cause Possibilities:**
- Unexpected costs (shipping, taxes)
- Complex checkout process
- Limited payment options
- Security concerns

**Action Steps:**
1. Display total cost upfront → remove hidden fees → offer free shipping threshold
2. Reduce checkout to 1-2 steps → enable guest checkout → autofill forms
3. Add trust badges → display security certifications → offer multiple payment methods

**Expected Impact:** Reduce abandonment to 35-45% within 3 weeks
**Measurement:** Cart abandonment rate, checkout completion rate, revenue per visitor

### IF Sign-Up Conversion <5% THEN

**Diagnosis Criteria:**
- Low visitor-to-lead conversion
- Form abandonment high
- CTA click rate low

**Root Cause Possibilities:**
- Friction in sign-up process
- Weak value proposition
- Trust deficit
- Too many form fields

**Action Steps:**
1. Reduce form fields to 2-3 essential only → test email-only option → measure completion rate
2. Add social proof above form → display user count/testimonials → build trust
3. A/B test CTA copy → emphasize benefit over action → test button colors/placement

**Expected Impact:** Increase conversion to 8-12% within 2 weeks
**Measurement:** Form completion rate, field abandonment points, CTA click-through rate

## Revenue Optimization

### IF AOV Stagnant THEN

**Diagnosis Criteria:**
- Average order value unchanged for 3+ months
- No growth in revenue per customer
- Upsell/cross-sell rates low

**Root Cause Possibilities:**
- No bundling strategy
- Weak product recommendations
- Shipping thresholds not optimized
- Lack of premium options

**Action Steps:**
1. Create product bundles → offer 15% discount → position as "complete solution"
2. Implement "Frequently bought together" → test algorithmic vs manual curation → measure attachment rate
3. Set free shipping at 30% above current AOV → display progress bar → track threshold achievement rate

**Expected Impact:** 20-35% AOV increase within 4 weeks
**Measurement:** Average order value, units per transaction, bundle adoption rate

### IF Refund Rate >5% THEN

**Diagnosis Criteria:**
- High return/refund requests
- Customer dissatisfaction signals
- Negative reviews mentioning quality

**Root Cause Possibilities:**
- Product-market misalignment
- Misleading marketing/descriptions
- Quality control issues
- Poor customer expectations setting

**Action Steps:**
1. Analyze refund reasons → categorize top 3 causes → address systematically
2. Enhance product descriptions → add detailed specs/measurements → include user-generated photos
3. Implement quality checks → update supplier standards → test product samples pre-launch

**Expected Impact:** Reduce refund rate to <3% within 6 weeks
**Measurement:** Refund rate, return reason categories, customer satisfaction score

### IF Customer LTV Declining THEN

**Diagnosis Criteria:**
- Repeat purchase rate dropping
- Customer lifespan shortening
- Retention cohorts weakening

**Root Cause Possibilities:**
- Poor post-purchase experience
- Lack of retention strategy
- Product quality inconsistency
- Weak customer relationships

**Action Steps:**
1. Launch email nurture sequence → send value content → offer exclusive deals → track engagement
2. Create loyalty/rewards program → reward repeat purchases → measure participation rate
3. Survey churned customers → identify pain points → implement fixes → win-back campaign

**Expected Impact:** 25-40% LTV increase within 12 weeks
**Measurement:** Repeat purchase rate, customer lifespan, cohort retention curves

## Content Optimization

### IF Video Completion Rate <50% THEN

**Diagnosis Criteria:**
- Viewers drop off before halfway
- Average watch time <50% of duration
- High early exit rate

**Root Cause Possibilities:**
- Weak hook/intro
- Content too long for platform
- Pacing issues/boring segments
- Value delivered too late

**Action Steps:**
1. Front-load value → deliver key insight in first 10 seconds → tease outcome early
2. Cut video length by 30% → remove filler → test shorter formats → measure completion lift
3. Add pattern interrupts every 15 seconds → change scene/angle → use text overlays → maintain attention

**Expected Impact:** Increase completion rate to 60-75% within 2 weeks
**Measurement:** Video completion rate, avg watch percentage, audience retention graph

### IF Save/Share Rate Low (<3%) THEN

**Diagnosis Criteria:**
- Content consumed but not shared
- Low viral coefficient
- Limited organic amplification

**Root Cause Possibilities:**
- Content not "shareable" (too personal/generic)
- No clear value for saving
- Missing share triggers
- Weak emotional resonance

**Action Steps:**
1. Create reference content → checklists/templates/frameworks → make instantly valuable
2. Add explicit share CTA → "Tag someone who needs this" → test different prompts → measure share rate
3. Evoke emotion → use storytelling → create "I relate" moments → inspire identity-based sharing

**Expected Impact:** Increase save/share rate to 8-12% within 3 weeks
**Measurement:** Save rate, share rate, virality coefficient

### IF Comment Quality Poor THEN

**Diagnosis Criteria:**
- Generic comments ("Great post!")
- Low conversation depth
- Engagement rate high but shallow

**Root Cause Possibilities:**
- Content not discussion-worthy
- No questions posed
- Community not engaged
- Topic not controversial/interesting enough

**Action Steps:**
1. End posts with specific question → avoid yes/no questions → ask for experiences/opinions
2. Reply to every comment within 2 hours → ask follow-up questions → build conversation threads
3. Create polarizing (but respectful) takes → present both sides → invite debate → moderate actively

**Expected Impact:** 3-5x improvement in comment depth within 2 weeks
**Measurement:** Comments per post, comment word count, reply rate

## Implementation Framework

### Prioritization Matrix
1. **Quick Wins:** High impact, low effort → implement immediately
2. **Strategic Bets:** High impact, high effort → plan for quarterly execution
3. **Fill-Ins:** Low impact, low effort → delegate or automate
4. **Time Sinks:** Low impact, high effort → eliminate

### Testing Protocol
1. **Baseline:** Measure current state (1 week)
2. **Hypothesis:** Define expected outcome
3. **Test:** Run for minimum 2 weeks
4. **Analyze:** Compare to baseline
5. **Scale or Kill:** 20%+ improvement → scale, <20% → kill

### Success Metrics Dashboard
- **Traffic:** Weekly reach, impressions, traffic sources
- **Engagement:** Engagement rate, watch time, save/share rate
- **Conversion:** Lead conversion, checkout completion, bounce rate
- **Revenue:** AOV, LTV, refund rate, repeat purchase rate

---

**Usage:** Identify your symptom → follow IF-THEN pattern → implement action steps → measure results → iterate.
