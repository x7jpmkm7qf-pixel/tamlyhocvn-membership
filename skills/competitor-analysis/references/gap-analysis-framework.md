# Market Gap Analysis Framework

## Gap Identification Methodology

### Step 1: Map Competitor Landscape
- **Direct competitors**: Same product/service, same audience
- **Indirect competitors**: Different product, same need/outcome
- **Substitute competitors**: Alternative solutions to same problem

### Step 2: Positioning Map (2x2 Matrix)
**Price vs Quality Matrix:**
- High Price/High Quality (Premium)
- High Price/Low Quality (Overpriced)
- Low Price/High Quality (Value)
- Low Price/Low Quality (Budget)

**Features vs Simplicity Matrix:**
- Complex/Feature-Rich (Power users)
- Complex/Limited Features (Confusing)
- Simple/Feature-Rich (Sweet spot)
- Simple/Limited Features (Basic)

### Step 3: Competitive Advantage Discovery Matrix

| Price point | | | | |

| Product features | | | | |

Rate each: 1 (Weak) to 5 (Strong)

### Step 4: Identify Underserved Segments
- Demographics: age, location, income, education
- Psychographics: values, interests, lifestyle
- Behavioral: usage patterns, buying habits
- Pain points not addressed by current solutions

## Gap Categories

### Content Gaps
**What competitors don't cover:**
- Niche topics within main category
- Vietnamese-specific case studies
- Beginner-friendly tutorials (or advanced deep-dives)
- Video format when everyone uses text
- Local language content in English-dominated niches

**Exploitation Strategy:**
- Create comprehensive content in underserved areas
- Become the go-to resource for specific subtopics
- Use SEO to dominate gap keywords

### Offer Gaps
**Products/services not available:**
- Mid-tier pricing (between cheap and premium)
- Bundled solutions (competitors sell separately)
- Customization options competitors don't offer
- Subscription model in one-time purchase market (or vice versa)
- Done-for-you service in DIY market

**Exploitation Strategy:**
- Test MVP with small audience first
- Position as "the only" solution offering X
- Create comparison content highlighting uniqueness

### Service Gaps
**Support/experience shortcomings:**
- Slow response times (24+ hours)
- No Vietnamese language support
- Lack of onboarding/training
- Poor post-purchase experience
- No community/peer support

**Exploitation Strategy:**
- Guarantee faster response (1-hour, same-day)
- Offer bilingual support (VN + EN)
- Create comprehensive onboarding process
- Build community as competitive moat

### Price Gaps
**Underserved price points:**
- Ultra-budget option missing (under 100k VND)
- Premium tier absent (over 10M VND)
- Payment flexibility lacking (installments, trials)
- Hidden costs competitors charge

**Exploitation Strategy:**
- Introduce transparent pricing at gap point
- Offer payment plans competitors don't
- Remove unnecessary costs to hit sweet spot
- Create premium option with exclusive benefits

### Speed Gaps
**Slow delivery competitors:**
- 7+ day delivery in fast-paced market
- Slow content updates (monthly vs weekly)
- Delayed customer onboarding
- Lengthy implementation timelines

**Exploitation Strategy:**
- Guarantee faster delivery (same-day, instant)
- Streamline processes for quick wins
- Automate onboarding for immediate access
- Market speed as key differentiator

## Vietnamese Market Considerations

### Local Gaps Common in VN Market:
- Payment methods: Cash on delivery, installments, e-wallets (Momo, ZaloPay)
- Language: Full Vietnamese support, cultural context
- Pricing: VND-based pricing psychology (990k vs 1M)
- Mobile-first: Desktop competitors in mobile market
- Social platforms: Zalo integration, Facebook-focused
- Trust building: Strong social proof needs (testimonials, community)
- Shipping: Nationwide coverage vs Hanoi/HCMC only

### Cultural Service Gaps:
- Personal touch in automated market
- Family/group buying options
- Face-to-face consultation in digital market
- Long-term relationship vs transactional
- Education-first approach (free value before selling)

## Exploitation Strategy Framework

**For Each Identified Gap:**

1. **Validate Gap is Real**
   - Survey target audience (10+ conversations)
   - Check search volume for gap-related keywords
   - Test small offer/content to gauge interest

2. **Assess Your Ability to Fill Gap**
   - Skills/resources required
   - Time to implement
   - Cost to deliver
   - Sustainable advantage

3. **Positioning Strategy**
   - How to communicate this difference
   - Marketing angles to emphasize gap
   - Proof points needed

4. **Launch Plan**
   - MVP/beta version (2-4 weeks)
   - Initial audience (warm list, partners)
   - Feedback loop for refinement
   - Scale plan if validated

5. **Competitive Moat**
   - Why competitors can't copy easily
   - Network effects to build
   - Expertise barriers
   - Speed to market advantage

## Red Flags (Fake Gaps)

Avoid gaps that are:
- Too small (less than 100 people market)
- Unprofitable (can't charge enough)
- Temporary trends (fleeting demand)
- High competition incoming (obvious opportunity)
- Misaligned with your strengths

## Quarterly Gap Review

Every 3 months:
- Reassess competitive landscape (new entrants?)
- Check if gaps are closing (competitors adapting?)
- Find new emerging gaps (market shifts?)
- Adjust positioning based on gap changes
