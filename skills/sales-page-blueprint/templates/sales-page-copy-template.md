# Sales Page Copy Template

Complete fill-in-the-blank framework for high-converting sales pages.

## Hero Section

### Headline Variations

**Option 1: Transformation + Timeframe**
```
[ACTION VERB] [DESIRED OUTCOME] in [SPECIFIC TIMEFRAME]
without [BIGGEST PAIN POINT]
```
Example: "Build a Profitable Online Business in 90 Days without Technical Skills"

**Option 2: Question Hook**
```
What if you could [DESIRED OUTCOME] without [COMMON STRUGGLE]?
```
Example: "What if you could attract premium clients without cold calling?"

**Option 3: Bold Promise**
```
The [ADJECTIVE] way to [ACHIEVE GOAL] (even if [OBJECTION])
```
Example: "The proven system to scale your agency (even if you're a solopreneur)"

## Quick Fill Instructions

1. **Replace all [BRACKETED ITEMS]** with your specific information
2. **Choose one headline variation** that best fits your offer
3. **Select 5-7 benefit bullets** from the templates (don't use all)
4. **Pick 3 testimonials** that represent different customer segments
5. **Add 3-4 bonuses** that complement (not duplicate) core offer
6. **Use 2 pricing options minimum** (payment plan + full payment)
7. **Include 5-8 FAQs** covering top objections
8. **Choose soft or hard close** based on your brand voice
9. **Add urgency element** if legitimate (deadline, limited spots, etc)
10. **Keep mobile readers in mind** — short paragraphs, clear CTAs

## Customization Notes

**For High-Ticket Offers** (over $1000):
- Add credibility section with your background
- Include longer-form testimonials (video preferred)
- Add application/qualification section
- Emphasize personalization and support
- Consider adding FAQ about payment plans

**For Low-Ticket Offers** (under $100):
- Simplify page (shorter overall)
- Focus on quick wins and speed
- More urgency elements
- Less social proof needed
- Emphasize easy implementation

**For Service-Based Offers**:
- Add process/timeline section
- Show example deliverables
- Include your working style/approach
- Add portfolio or case studies
- FAQ about communication and revisions

**For Course/Educational Products**:
- Add curriculum/module breakdown
- Show lesson count and video hours
- Include learning outcomes
- Add student success gallery
- FAQ about access and platform

**For Software/Tools**:
- Add features section with screenshots
- Include integration list
- Show dashboard/interface
- Add technical specs
- FAQ about setup and support
