## USAGE GUIDELINES

### Density Rules
- Max 2-3 power words per sentence
- Don't stack multiple strong words (overwhelming)
- Balance power words with plain language
- Use strategically in headlines, subheads, CTAs

### Testing Protocol
1. Write baseline copy (moderate power words)
2. Create variation with 30% more power words
3. Create variation with emotional triggers emphasized
4. A/B test all three versions
5. Measure: CTR, conversion rate, engagement time

### Context Matters
- **B2B:** Authority + Trust + Practical value
- **B2C:** Desire + Urgency + Exclusivity
- **High-ticket:** Trust + Authority + Transformation
- **Low-ticket:** Urgency + Curiosity + Quick wins
- **Info products:** Curiosity + Transformation + Authority

## POWER WORD INTENSITY SCALE

### Level 1: Subtle (Trust-Building)
Proven, reliable, effective, practical, clear

### Level 2: Moderate (Engagement)
Powerful, exclusive, advanced, comprehensive, breakthrough

### Level 3: Strong (Conversion)
Ultimate, explosive, revolutionary, dominate, transformation

### Level 4: Maximum (Use Sparingly)
Secrets, forbidden, shocking, insane, mind-blowing

**Rule:** Use Level 1-2 for 80% of copy. Reserve Level 3-4 for key moments (headlines, CTAs, pivots).
