# Pricing Psychology Tactics for Vietnamese Market

## Overview
Strategic pricing techniques designed for Vietnamese digital product creators to maximize perceived value and conversion rates.

---

## 1. Anchoring Effect Strategy

### High-to-Low Presentation
Present premium option first to set mental benchmark:
- VIP Package: 2,997,000 VND
- Standard Package: 997,000 VND (feels like a bargain)
- Basic Package: 497,000 VND

### Reference Price Display
```
Giá gốc: 1,500,000 VND (crossed out)
Giá hôm nay: 997,000 VND (emphasized)
Tiết kiệm: 503,000 VND (33% off)
```

---

## 2. Charm Pricing for Vietnamese Market

### Number Psychology in VND
- **497,000** — Feels under 500K threshold
- **789,000** — Lucky number 8, under 800K
- **997,000** — Under 1M psychological barrier
- **1,497,000** — Premium but under 1.5M
- **2,997,000** — High-end under 3M

### Avoid These Numbers
- Round numbers (500K, 1M, 2M) — seem arbitrary
- Number 4 in final digits (Vietnamese superstition)

## 4. Bundle vs Unbundled Psychology

### Bundling Wins When:
```
Unbundled perception:
Course: 500K + Templates: 200K + Coaching: 500K = 1,200K

Bundle offer:
Complete Package: 797,000 VND
(Tiết kiệm 403,000 VND!)
```

### When to Unbundle:
- Low-ticket products (under 300K)
- Testing market demand
- Building product ladder

## 10. Testing and Optimization

### A/B Test These Elements:
- Price points (797K vs 799K vs 749K)
- Payment terms (1-pay vs 3-pay)
- Tier naming (Basic/Pro/Premium vs Starter/Growth/Scale)
- Value stack presentation

### Metrics to Track:
- Conversion rate by tier
- Average order value
- Cart abandonment rate
- Installment vs one-time ratio

### Adjustment Signals:
- <5% conversion: Price too high or value unclear
- >30% choosing premium: Basic too weak
- <10% installment uptake: Fee too high

## Vietnamese Market Considerations

### Income Brackets
- Entry-level: 297-497K (students, beginners)
- Mid-market: 697-1,497K (working professionals)
- Premium: 1,997K+ (business owners, agencies)

### Payment Platform Preferences
1. MoMo (most popular)
2. Banking transfer (trusted for high-ticket)
3. ZaloPay (younger audience)
4. Shopee Pay (e-commerce buyers)
5. Credit card (premium buyers)

### Cultural Pricing Notes
- Vietnamese value "giảm giá" heavily
- Show original price even if inflated
- Lucky numbers: 8, 9, 6
- Avoid number 4 (death association)
- Group buying mentality (offer referral discounts)

## Quick-Start Pricing Formula

```
1. Calculate base cost (time + expertise value)
2. Apply 3x multiplier for profit margin
3. Round to charm number (497K, 797K, 997K)
4. Create 3 tiers (base, base×2.5, base×6)
5. Add value stack showing 2-3x perceived value
6. Test with early bird discount (30% off)
7. Measure and adjust monthly
```

**Example:**
- Base cost: 300K → 3x = 900K
- Round to: 897K
- Tiers: 297K, 897K, 2,997K
- Launch at: 697K (early bird)
